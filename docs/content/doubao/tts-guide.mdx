title: 豆包 TTS 使用指南
---

本文介绍如何使用 univoice SDK 中的豆包 TTS（文字转语音）功能。

## 概述

DoubaoTTS 基于火山引擎 TTS 双向流式协议，使用 **WebSocket** 实现语音合成，支持两种合成模式：

| 模式 | 方法 | 说明 | 适用场景 |
|------|------|------|----------|
| 非流式 | `synthesize` | 一次性返回完整音频 | 文件合成、离线处理 |
| 流式 | `speak` + `stream: true` | 边合成边接收音频块 | 实时播报、LLM 流式输出转语音 |

提供两个模型版本：

| 版本 | Resource ID | 说明 |
|------|-------------|------|
| V2（默认） | `seed-tts-2.0` | 性能更优，推荐用于一般场景 |
| V1（兼容） | `seed-tts-1.0` | 兼容旧版，适用于遗留系统 |

## 创建 TTS 实例

univoice 提供两种创建 DoubaoTTS 实例的方式：

<Tabs items={['工厂函数（推荐）', '直接实例化']}>
<Tab>

使用 `createTTS` 工厂函数，需先导入 `univoice/tts/providers` 注册提供商：

```typescript
import 'univoice/tts/providers';
import { createTTS } from 'univoice/tts';

const tts = createTTS({
  provider: 'doubao',
  appId: process.env.DOUBAO_APP_ID,
  accessToken: process.env.DOUBAO_ACCESS_TOKEN,
  voice: 'zh_female_tianmeixiaoyuan_moon_bigtts',
  format: 'mp3',
});
```

</Tab>
<Tab>

直接导入 `DoubaoTTS` 类并实例化，无需注册 provider：

```typescript
import { DoubaoTTS } from 'univoice/tts/providers';

const tts = new DoubaoTTS({
  appId: process.env.DOUBAO_APP_ID,
  accessToken: process.env.DOUBAO_ACCESS_TOKEN,
  voice: 'zh_female_tianmeixiaoyuan_moon_bigtts',
  format: 'mp3',
});
```

</Tab>
</Tabs>

<Callout type="info">
两种方式创建的实例拥有完全一致的 API，推荐使用工厂函数，便于切换不同提供商。
</Callout>

## 非流式合成

最简单的使用方式 —— 传入文本，一次性获取完整音频：

```typescript
import 'univoice/tts/providers';
import { createTTS } from 'univoice/tts';

const tts = createTTS({
  provider: 'doubao',
  appId: process.env.DOUBAO_APP_ID,
  accessToken: process.env.DOUBAO_ACCESS_TOKEN,
  voice: 'zh_female_tianmeixiaoyuan_moon_bigtts',
  format: 'mp3',
});

// 传入文本，返回完整音频
const response = await tts.synthesize({
  text: '欢迎来到杭州！我是您的智能导游。',
});

console.log('音频大小:', response.audio.length, 'bytes');
console.log('音频格式:', response.format);
```

### TTSResponse 结构

| 字段 | 类型 | 说明 |
|------|------|------|
| `audio` | `Buffer \| Uint8Array` | 音频数据 |
| `format` | `string` | 音频格式（如 `mp3`、`pcm`） |
| `duration` | `number?` | 音频时长 |

## 流式合成

流式合成适合实时场景，文本边发送边接收音频块，也可接收 LLM 流式文本输出：

```typescript
import 'univoice/tts/providers';
import { createTTS } from 'univoice/tts';

const tts = createTTS({
  provider: 'doubao',
  appId: process.env.DOUBAO_APP_ID,
  accessToken: process.env.DOUBAO_ACCESS_TOKEN,
  voice: 'zh_female_tianmeixiaoyuan_moon_bigtts',
  format: 'pcm',
  sampleRate: 24000,
});

// 传入字符串，流式接收音频块
for await (const { audioChunk } of tts.speak('欢迎来到杭州！', { stream: true })) {
  console.log('收到音频块:', audioChunk.length, 'bytes');
}
```

也支持传入文本流（如 LLM 流式输出）：

```typescript
import 'univoice/tts/providers';
import { createTTS } from 'univoice/tts';

const tts = createTTS({
  provider: 'doubao',
  appId: process.env.DOUBAO_APP_ID,
  accessToken: process.env.DOUBAO_ACCESS_TOKEN,
  voice: 'zh_female_tianmeixiaoyuan_moon_bigtts',
  format: 'pcm',
  sampleRate: 24000,
});

// 模拟 LLM 流式输出
async function* textStream() {
  yield '欢迎来到杭州！';
  yield '我是您的智能导游。';
}

// 流式文本输入 → 流式音频输出
for await (const { audioChunk } of tts.speak(textStream(), { stream: true })) {
  console.log('收到音频块:', audioChunk.length, 'bytes');
}
```

### TTSStreamChunk 结构

| 字段 | 类型 | 说明 |
|------|------|------|
| `audioChunk` | `Uint8Array` | 音频数据块 |

## 模型版本对比

| 特性 | `seed-tts-1.0` | `seed-tts-2.0` |
|------|----------------|----------------|
| 协议版本 | V1 | V2 |
| 默认状态 | — | 默认使用 |
| 性能 | 标准性能 | 性能更优 |
| 适用场景 | 兼容旧版系统 | 一般场景（推荐） |

通过 `resourceId` 参数切换模型版本：

```typescript
const tts = createTTS({
  provider: 'doubao',
  appId,
  accessToken,
  resourceId: 'seed-tts-1.0', // 使用 V1
  // resourceId: 'seed-tts-2.0', // 使用 V2（默认）
});
```

## 音频格式支持

DoubaoTTS 支持以下音频格式，通过 `format` 参数指定：

<Tabs items={['MP3', 'PCM', 'WAV', 'OGG Opus']}>
<Tab>

MP3 是最常用的音频格式，兼容性好，文件体积小：

```typescript
const tts = createTTS({
  provider: 'doubao',
  appId,
  accessToken,
  format: 'mp3',
});

const response = await tts.synthesize({ text: '你好世界' });
// response.audio 为 MP3 数据，可直接保存为 .mp3 文件
```

</Tab>
<Tab>

PCM 是原始音频数据，无压缩、无容器封装，延迟最低：

```typescript
const tts = createTTS({
  provider: 'doubao',
  appId,
  accessToken,
  format: 'pcm',
  sampleRate: 24000,
});

const response = await tts.synthesize({ text: '你好世界' });
// response.audio 为 PCM 原始数据
```

<Callout type="info">
PCM 格式不包含文件头信息，播放时需要指定采样率、位深和声道数。默认配置为 24000Hz、16bit、单声道。
</Callout>

</Tab>
<Tab>

WAV 格式包含文件头，可直接播放：

```typescript
const tts = createTTS({
  provider: 'doubao',
  appId,
  accessToken,
  format: 'wav',
  sampleRate: 24000,
});
```

</Tab>
<Tab>

OGG Opus 格式压缩率高，适合流式传输场景：

```typescript
const tts = createTTS({
  provider: 'doubao',
  appId,
  accessToken,
  format: 'ogg_opus',
  sampleRate: 24000,
});
```

</Tab>
</Tabs>

## 完整配置参考

### DoubaoTTSOptions

#### 认证配置

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `appId` | `string` | `''` | 火山引擎 App ID |
| `accessToken` | `string` | `''` | 火山引擎 Access Token |
| `resourceId` | `string` | `'seed-tts-2.0'` | 火山引擎 Resource ID（模型版本） |

#### 合成配置

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `voice` | `string` | `'zh_female_tianmeixiaoyuan_moon_bigtts'` | 发音人音色 |
| `format` | `'mp3' \| 'wav' \| 'ogg' \| 'flac' \| 'pcm' \| 'opus' \| 'ogg_opus'` | `'mp3'` | 音频格式 |
| `sampleRate` | `number` | `24000` | 采样率 |
| `speed` | `number` | `1.0` | 语速倍率 |
| `volume` | `number` | `1.0` | 音量倍率 |
| `pitch` | `number` | `1.0` | 音调倍率 |
| `enableTimestamp` | `boolean` | `false` | 是否启用时间戳 |

#### 通用配置

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `baseUrl` | `string` | `'wss://openspeech.bytedance.com/api/v3/tts/bidirection'` | WebSocket 服务地址 |
| `model` | `string` | `'default'` | 模型名称 |
| `language` | `string` | `'zh-CN'` | 语言 |

## 工具函数

### collectAudio

收集音频数据，支持回调：

```typescript
import { collectAudio } from 'univoice/tts';

const response = await tts.synthesize({ text: '你好世界' });
const audio = await collectAudio(response, {
  onComplete: (audio) => console.log('音频大小:', audio.length),
});
```

### saveTTSResponse

将 TTS 响应保存到文件：

```typescript
import { saveTTSResponse } from 'univoice/tts';

const response = await tts.synthesize({ text: '你好世界' });

// 自动生成文件名
const filepath = await saveTTSResponse(response);

// 指定文件名和目录
await saveTTSResponse(response, {
  filename: 'output.mp3',
  directory: './audio',
});
```

### saveAudio

将音频块保存到文件，支持自动添加 WAV 头：

```typescript
import { saveAudio } from 'univoice/tts';

// 保存流式音频块
const stream = tts.speak('你好世界', { stream: true });
await saveAudio('output.wav', stream, { sampleRate: 24000 });
```

### playAudio

播放音频（需要系统安装播放器，macOS 默认使用 `afplay`）：

```typescript
import { playAudio } from 'univoice/tts';

const response = await tts.synthesize({ text: '你好世界' });
await playAudio(response);

// 指定播放器
await playAudio(response, { player: 'ffplay' });
```

### teeAudio

同时保存和播放音频：

```typescript
import { teeAudio } from 'univoice/tts';

const response = await tts.synthesize({ text: '你好世界' });
await teeAudio(response, {
  save: { filename: 'output.mp3', directory: './audio' },
  play: { player: 'afplay' },
});
```

## 常见问题

### 非流式和流式模式如何选择？

- **非流式**（`synthesize`）：传入完整文本，一次性获取音频。适合文件合成、离线处理
- **流式**（`speak` + `stream: true`）：边发送边接收，实时获取音频块。适合实时播报、LLM 流式输出转语音

### 如何选择 `seed-tts-1.0` 和 `seed-tts-2.0`？

- **`seed-tts-2.0`**（默认）：性能更优，推荐用于一般场景
- **`seed-tts-1.0`**：仅用于兼容旧版系统

通过 `resourceId` 参数切换：

```typescript
const tts = createTTS({
  provider: 'doubao',
  appId,
  accessToken,
  resourceId: 'seed-tts-1.0', // 或 'seed-tts-2.0'
});
```

### 如何获取火山引擎凭证？

1. 前往 [火山引擎控制台](https://console.volcengine.com/speech/service/overview)
2. 开通「语音合成」服务
3. 在「应用管理」中创建应用，获取 App ID 和 Access Token

### PCM 格式音频如何播放？

PCM 是原始音频数据，需要指定参数播放：

```bash
ffplay -autoexit -f s16le -ar 24000 -ac 1 output.pcm
```

或者使用 `saveAudio` 保存为 WAV 格式（自动添加文件头）：

```typescript
import { saveAudio } from 'univoice/tts';

const response = await tts.synthesize({ text: '你好世界' });
// 保存为 WAV 文件，自动添加文件头
await saveAudio('output.wav', [response.audio], { sampleRate: 24000 });
```

### speak 方法支持哪些输入类型？

`speak` 方法的 `input` 参数支持：

- **字符串**：直接传入文本，适合固定文本的流式合成
- **AsyncIterable\<string\>**：流式文本输入，适合 LLM 流式输出转语音
- **OpenAI Stream**：直接接收 OpenAI SDK 的流式输出
